## Scope

Per arch spec §7.3 Pass #14: Render depth into shadow atlas tiles per light.
- **Per-light render**: Set viewport/scissor to light's atlas tile. Render shadow casters with depth-only PSO (depthTest=GreaterOrEqual, depthWrite=true, no color, front-face cull, depthBias).
- **Tier dispatch**: T1 uses mesh shader depth-only pipeline (same GpuCullPipeline from Phase 6a with `depthOnly=true`). T2/3/4 uses vertex shader + `DrawIndexedIndirect`.
- **Point lights**: 6 faces (cube map layout in atlas). Each face = separate viewport.
- **Spot lights**: 1 face, perspective projection from light position.
- **Area lights**: 1 face, orthographic projection for directional shadow component.
- **Budget**: <2ms for 32 shadow-casting lights (T1). Only dirty lights re-render (static scene optimization).
