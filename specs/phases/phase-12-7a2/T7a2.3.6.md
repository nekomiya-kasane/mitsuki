## Scope

Per arch spec §(5) Selectability Enforcement:

Two-layer enforcement (GPU reduce + CPU safety net):
- **GPU layer**: Collect shader / Drill Stage 1 checks `if ((instance.flags & selectableMask) == 0) return;` before atomic append. Push constant `selectableMask` (16-bit, 1 bit per layer). Prevents non-selectable hits from entering HitBuffer.
- **CPU layer**: `ApplySelectionFilter()` re-checks selectability as defense-in-depth.
- **Push constant update**: `selectableMask` recomputed when layer selectability changes (`CadScene::SetLayerSelectable()`). Static scenes: zero per-frame cost.
