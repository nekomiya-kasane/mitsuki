## Scope

Per arch spec §3 Pass #56: AMD FidelityFX CAS — single compute pass post-TAA/FXAA. All tiers. <0.2ms @4K.

- **Algorithm**: Per-pixel adaptive sharpening. Computes local contrast from 3×3 neighborhood. Applies sharpening proportional to inverse contrast (low contrast = more sharpening, high contrast = less to avoid ringing).
- **Implementation**: Single compute dispatch, 1 thread per pixel. Input: post-AA LDR color. Output: sharpened LDR color (in-place or separate target).
- **Push constant**: `{sharpness: float}` (0.0 = off, 1.0 = max). Default 0.5.
- **Conditional**: Always-on post-AA pass (near-zero cost). Can be disabled via push constant sharpness=0.
