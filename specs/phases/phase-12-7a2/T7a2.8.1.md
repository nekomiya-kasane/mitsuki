None (L0 task).

## Scope

Per arch spec §3 Pass #48 and roadmap:

- **Algorithm**: Hi-Z ray march in screen-space. Start from GBuffer hit point, trace along reflection direction using hierarchical depth stepping (mip-level acceleration). On hit: sample HDR color at hit UV.
- **Half-res trace**: Trace at half resolution (960×540 @4K), bilateral upsample to full-res using depth+normal edge detection. Saves 75% ray cost.
- **Roughness gate**: Skip pixels with roughness > 0.5 (too blurry for screen-space, IBL cubemap sufficient).
- **Temporal accumulation**: Reproject previous frame SSR result via motion vectors. Blend with current frame via exponential moving average. Reduces noise from single-sample tracing.
- **Fallback**: When SSR miss (off-screen or too rough), fall back to IBL specular cubemap.
- **Budget**: <1.5ms @4K (half-res). Tier1/2 only.

### Technical Direction

- Hi-Z ray march: start at mip 0, step along ray. If depth test fails (behind surface), go up one mip (larger step). If passes (in front), go down one mip (finer step). Converges in ~20-30 steps.
- Per-pixel jitter: blue noise offset to break banding artifacts across frames.
- Contact hardening: weight by distance from hit to origin — closer hits are sharper.
