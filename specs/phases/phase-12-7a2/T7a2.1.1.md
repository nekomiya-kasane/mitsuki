- `OitNode`: `{float4 color, float depth, uint next}` = 24B → pack to 16B: `{R16G16B16A16F color (8B), float depth (4B), uint next (4B)}`.
- `OitNodePool`: SSBO of `OitNode[maxNodes]`. Default 16M nodes = 256MB.
- `OitHeadImage`: R32_UINT storage image (viewport resolution). Each pixel stores index of linked-list head. Sentinel = 0xFFFFFFFF.
- `OitConfig`: `{maxNodes, nodePoolUsageThreshold(80%), shrinkThreshold(30%), maxLayersPerPixel(16)}`.
- **Adaptive pool**: grow when usage > 80%, shrink when < 30%. Monitored via atomic counter.
