## Dependencies

None (L0 task).

## Scope

Per arch spec §3 Pass #58: Sobel on depth + normal discontinuities → edge mask → configurable outline color. All tiers. <0.2ms. Separate from GPU HLR (Phase 7a-1) — this is a screen-space approximation for illustration style, not topology-accurate edge detection.

- **Algorithm**: 3×3 Sobel filter on depth buffer + 3×3 Sobel on GBuffer normal. Edge = `max(depthEdge, normalEdge) > threshold`. Output: RGBA8 outline color where edge detected, transparent elsewhere.
- **Push constants**: `{outlineColor: float4, depthThreshold: float, normalThreshold: float}`.
- **Integration**: Fragment pass, reads depth (#1) + normals (#10). Composited via LayerStack Compositor (#65) as Overlay layer.
- **Use case**: Quick silhouette visualization without full HLR pipeline overhead. Active in Pen/Artistic/Sketchy display styles alongside HLR, or standalone for fast preview.
