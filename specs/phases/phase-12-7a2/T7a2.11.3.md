# T7a2.11.3 — Chromatic Aberration (3-Channel Radial Distortion)

**Phase**: 12-7a2
**Component**: Post-Process Bundle
**Status**: Not Started
**Effort**: L (< 1h)

## Dependencies

None (L0 task).

## Scope

Per arch spec §3: 3-channel radial distortion folded into tone-map pass. 2 extra texture fetches. Tier1/2. ~0 extra cost (folded into existing tone-map fragment shader).

- **Algorithm**: Sample R, G, B channels at slightly different UV offsets based on radial distance from screen center. `offset = normalize(uv - 0.5) * radialDistance² * caStrength`. R = sample at `uv + offset * 1.0`, G = sample at `uv` (no offset), B = sample at `uv - offset * 1.0`.
- **Push constant**: `{caStrength: float}` (0.0 = off, default 0.002).
- **Integration**: Folded into existing ToneMapping fragment shader (Phase 3b). 2 extra texture reads — negligible cost.
