## Dependencies

None (L0 task).

## Scope

GPU-computed explode transforms from assembly hierarchy. Per arch spec:
- **Assembly hierarchy buffer**: SSBO containing parent-child depth info per instance. Each instance has `assemblyLevel` (from `GpuInstance.flags[24:27]`).
- **Explode compute**: Per-instance displacement = `direction * distance * depthFactor^assemblyLevel`. Direction = centroid-to-instance vector (computed from AABB centroid). Multi-level: deeper assembly levels explode more.
- **Animation**: `smoothstep(t)` interpolation over configurable duration (default 500ms). Double-buffered transform: `worldMatrix_exploded = lerp(worldMatrix_base, worldMatrix_exploded_target, smoothstep(t))`.
- **Path visualization** (stretch): line from base position to exploded position, rendered as instanced line segments.

### Technical Direction

- Compute shader: 1 thread per instance, reads assembly hierarchy, writes exploded transform to a secondary transform buffer. SceneBuffer.worldMatrix remains camera-relative (RTE); explode offset added in world-space before RTE subtraction.
- Push constant: `{explodeFactor: float, animationT: float, centroid: float3}`.
- Zero CPU overhead in steady state (GPU-only transform update).
