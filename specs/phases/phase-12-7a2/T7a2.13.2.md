## Scope

5 built-in procedural material presets shipped with the engine, created using MaterialGraphCompiler:

1. **Brushed metal**: Perlin noise → anisotropic roughness direction, metallic=1.0, roughness=0.3-0.5.
2. **Anodized aluminum**: Voronoi noise → color variation, metallic=0.9, clearcoat=0.5.
3. **Carbon fiber**: Checker pattern (2-scale) → albedo weave + normal, metallic=0.0, roughness=0.4.
4. **Wood**: FBM noise → grain lines, albedo from gradient ramp (light→dark brown), roughness=0.6.
5. **Concrete**: Perlin noise → bump normal + roughness variation, albedo=gray, roughness=0.8.

Each preset = JSON material graph file + pre-compiled Slang permutation. Users can clone and modify.
