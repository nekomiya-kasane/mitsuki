## Scope

Per arch spec §3 Pass #50:

- **Algorithm**: Gather-based bokeh (Jimenez 2014 "Next Generation Post Processing in Call of Duty: Advanced Warfare").
- **Circle of Confusion (CoC)**: `CoC = abs(aperture * focalLength * (depth - focusDistance) / (depth * (focusDistance - focalLength)))`. Stored per-pixel as signed R16F (positive = far blur, negative = near blur).
- **Half-res gather**: Downsample CoC + color to half-res. 16 tap weighted gather per pixel. Kernel shape: disc (bokeh). Weight = `smoothstep(0, 1, sampleCoC / maxCoC)`.
- **Upsample**: Bilateral upsample using CoC edge detection.
- **Tier dispatch**: T1/T2 = compute shader. T3/T4 = simplified Gaussian blur (no bokeh shape).
- **Activation**: Only in Realistic display mode or explicit user request. Conditional RG node.
- **Budget**: <1.5ms @4K (half-res).

### Technical Direction

- CoC computation as separate compute pass (reusable by motion blur for velocity weighting).
- Bokeh shape: disc approximation via weighted Poisson disc sampling (16 taps). Phase 14 stretch: hexagonal bokeh via anisotropic filter.
- Near-field leak prevention: dilate near-field CoC before gather to prevent sharp near objects bleeding into far bokeh.
