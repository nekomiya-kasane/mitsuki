## Scope

No-drill area selection via VisBuffer mask + collect pipeline. Per arch spec Pick Architecture:
1. Generate selection mask (box = simple rect test; lasso = winding number compute, see T7a2.3.3).
2. Dispatch collect compute over mask region: for each pixel where mask==1, read VisBuffer `{instanceId, primitiveId}` → atomic-append to `HitBuffer` SSBO.
3. Dedup via `GpuRadixSort` (Phase 6a) by instanceId → `GpuCompact` → readback unique instance list.

Cost dominated by mask size, not scene complexity. <1ms for box, <1.5ms for lasso @4K.
