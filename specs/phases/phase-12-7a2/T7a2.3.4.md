## Scope

CPU post-processing filter applied after GPU pick + dedup + readback. Per arch spec §(6) SelectionFilter:

```cpp
struct SelectionFilter {
    PickLevel                level;               // topology level (required)
    std::optional<FaceType>  faceType;            // Planar|Cylindrical|Conical|Spherical|...
    uint16_t                 layerMask = 0xFFFF;  // only select from these layers
    bool                     selectableOnly = true;
    std::function<bool(const PickResult&)> customPredicate = nullptr;
};
```

Pipeline: `GPU pick → HitBuffer → dedup → readback → ApplyPickLevel(hits, level) → ApplySelectionFilter(hits, filter) → return`.

- **Layer mask**: `(instance.layerBits & filter.layerMask) != 0`.
- **FaceType**: query `TopoGraph::GetFaceType(entityId, faceId)`, compare. Only when `level >= Face`.
- **Custom predicate**: `std::function<bool(PickResult const&)>` — host application extension point.
- **CommandBus**: `selection.set_filter level:Face faceType:Cylindrical layerMask:0x000F`.
