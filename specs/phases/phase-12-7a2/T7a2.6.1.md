Per arch spec §7.1:

```cpp
struct GpuLight {                          // 64 bytes, alignas(16)
    float3   position;                     // 12B
    float    range;                        // 4B  → 16B
    float3   direction;                    // 12B
    float    innerConeAngle;               // 4B  → 32B (area: areaWidth)
    float3   color;                        // 12B
    float    outerConeAngle;               // 4B  → 48B (area: areaHeight)
    float    intensity;                    // 4B  (lumens for point/spot/area, lux for directional)
    uint32_t type;                         // 4B  (0=Dir,1=Point,2=Spot,3=AreaRect,4=AreaDisc,5=AreaTube)
    uint32_t shadowIndex;                  // 4B  (ShadowAtlas tile, 0xFFFFFFFF=none)
    uint32_t flags;                        // 4B  → 64B (castShadow|volumetric|animated)
};
static_assert(sizeof(GpuLight) == 64);
```

Physical units: Directional = lux (lx), Point/Spot/Area = lumens (lm). Matches Filament/UE5/DSPBR.
