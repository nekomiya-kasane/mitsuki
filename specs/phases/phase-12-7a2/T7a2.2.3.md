## Scope

Two pick paths per arch spec Pick Architecture:
- **No-drill**: VisBuffer single-pixel readback at click position → instanceId + primitiveId → TopoGraph resolve. Cost: ~0.05ms (1 pixel read + CPU lookup).
- **Drill**: `VK_KHR_ray_query` in compute shader. Single ray from camera through click pixel. Multi-hit: `rayQueryProceedEXT` loop collects all intersections along ray. Output: `HitBuffer` SSBO with depth-sorted hits. Cost: <0.5ms.
- **Async readback**: Timeline semaphore signals when GPU pick complete → CPU reads result without stalling render.

### Technical Direction

- Per arch spec: "VisBuffer single-pixel readback (0.05ms)" for no-drill, "RT ray query multi-hit through TLAS (<0.5ms)" for drill.
- Topology-level return: `{instanceId, meshletIdx, primitiveId}` → `TopoGraph::PrimitiveToFace()` → `{entityId, faceId, edgeIds[], vertexIds[]}`.
- `PickLevel` enum: `{Part, Body, Face, Edge, Vertex}` — truncates topology path.
