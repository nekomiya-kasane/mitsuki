## Design Decisions

- **SectionPlaneConfig (64B)**: Per-plane GPU struct with planeEquation, capColor, capMaterialType, booleanOp, refGeo settings. Matches HOOPS/OCCT per-section visual attributes.
- **SectionPlaneSet**: CPU-side collection with active bitmask. Not a GPU struct — `BuildSectionPlaneUBO()` converts to `SectionPlaneUBO` (528B) for upload.
- **SectionPlaneUBO (528B)**: 8 × 64B planes + activeMask + planeCount. Single UBO binding, updated only when planes change.
- **Boolean ops per-plane**: Each plane carries its own `booleanOp` (AND/OR). AND = any plane clips → discard. OR = all planes must clip → discard.
- **Max 8 planes**: Matches `GpuInstance.clipPlaneMask` uint8 bitmask (already at offset 104).
- **CPU clip helpers**: `IsClippedByPlane()` (constexpr) and `IsClippedBySet()` for testing and CPU fallback.
