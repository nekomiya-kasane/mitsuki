## Design Decisions

- **ISO 128 ratios**: Base unit = 1mm at scale 1.0. All 15 patterns use standard proportions from ISO 128-20:2022 Table 1.
- **Industry presets**: ANSI Y14.2 and DIN ISO 128 use scale=1.0; JIS B 0001 uses scale=0.7 (Japanese standard uses smaller patterns).
- **Custom registration**: Starts at index 16 (`kCustomBaseIndex`), max 64 total patterns. Returns `Result<uint32_t>` for error handling.
- **GPU upload**: Single `UploadToGpu()` call creates a `Storage|TransferDst` SSBO with all registered patterns.
