## Context Anchor

Fragment shader renders procedural cross-hatch fill on stencil cap faces. Reads `HatchPatternDesc` from SSBO indexed by material type (per-section `capMaterialType` from `SectionPlaneConfig`). Computes hatch lines in **object-space UV**: project `worldPos` onto section plane local 2D coordinate frame (plane tangent + bitangent), then apply `fract(dot(planeLocalUV, hatchDirection) / spacing)`. This anchors hatching to geometry — no swimming artifacts when camera moves (unlike screen-space approach). Supports two-direction cross-hatch (primary + secondary angle). Anti-aliased via `fwidth` derivative.
