## Design Decisions

- **Staging buffer pattern**: Fixed T7a1.1.1 bug where `WriteBuffer` was called on `GpuOnly` buffers (Vulkan cannot map device-local memory). All GPU uploads now use CpuToGpu staging + CopyBuffer.
- **Push constant scalar layout**: Fixed Slang `float3`/`uint3` alignment mismatch — shader push constants now use scalar fields (`cameraPosX/Y/Z`) matching C++ struct layout exactly (32B).
- **0-based vertex/face/edge IDs**: Test topology uses 0-based IDs to match packed position buffer indexing (shader fetches `vertexIndex * stride`).
- **Degenerate meshlet**: `coneCutoff = -128` ensures pre-cull never rejects edges, isolating classify logic for testing.
