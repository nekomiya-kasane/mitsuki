## Design Decisions

1. **EdgeDescriptor 32B (2x16B)**: Stores world-space positions (camera-relative RTE) + packed attributes + instanceId. Face indices (faceA/faceB) excluded — classify reads them from TopoGraph SSBO via BDA, not stored per-edge. paramT intervals stored in separate VisibilityInterval buffers (1 edge -> N intervals).
2. **LinePatternSegment 4B**: Fixed-point length in 0.01mm units (uint16, range 0-655.35mm). Smaller than 8B alternative — fits 16 segments in 64B (4 cache lines) instead of 128B.
3. **ISO 128 auto-mapping**: Compile-time constexpr switch tables for EdgeType -> LineType/LineWeight. Silhouette=Continuous/0.70mm, Construction=Dashed/0.18mm, etc.
4. **EdgeBuffer RAII**: Move-only, single-owner. ResetCount via WriteBuffer (not compute fill) — simpler for single uint32.
