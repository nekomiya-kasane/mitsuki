## Context Anchor

### This Task's Contract

**Produces**: `SectionPlane::Clip()` + `SectionPlane::RenderCap()` — multi-plane clipping with watertight stencil capping.

- **Clip**: Fragment shader `discard` when `dot(planeEquation, worldPos) < 0` for enabled planes. AND = all planes must pass; OR = any plane passes.
- **Stencil capping**: Two-pass stencil: (1) render front-faces → stencil increment, (2) render back-faces → stencil decrement. Where stencil != 0 = inside geometry = cap fill needed. Fullscreen quad samples stencil → fills with cap color/hatch.
- **Watertight**: No gaps at section boundary — stencil method guarantees pixel-accurate caps.
- Per arch spec §3 Pass #31: "Section Plane/Volume — Fragment (stencil) — Scene depth, clipPlaneMask — Clip + cap + ISO 128 hatch — <0.5ms".
