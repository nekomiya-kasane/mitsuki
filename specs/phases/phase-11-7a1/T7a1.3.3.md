## Design Decisions

- **Indirection UBO (LineStyleMap, 32B)**: Runtime-remappable `LineStyle[8] → LineType` table. Matches HOOPS pattern. Avoids shader recompilation for mapping changes.
- **Priority chain**: `instanceLineStyle > 0` → use LineStyleMap; `== 0` → use edge's auto-mapped LineType from EdgeDescriptor.packed.
- **Shader-side**: `resolveLineType()` in `hlr_common.slang` reads `extractLineStyle(instanceFlags)` then indexes into `LineStyleMap.mapping[]`.
- **CPU-side**: `ResolveLineType()` constexpr function in `EdgeTypes.h` for CPU validation and test.
