- **Error model**: `OcctKernel::Import()` wraps `STEPControl_Reader` errors → `ErrorCode::ImportFailed`. `Tessellate()` wraps `BRepMesh_IncrementalMesh` → `ErrorCode::TessellationFailed`.
- **Thread safety**: OCCT is NOT thread-safe. One OcctKernel per thread.
- **Invariants**: `OcctKernel` auto-registers as "occt" in `KernelFactory` when `MIKI_KERNEL_OCCT=ON`.

### Upstream Contracts

- T5.8.1: `IKernel` interface, `KernelFactory::Register()`, `TessellationResult`, `ImportResult`
  - Source: `include/miki/kernel/IKernel.h`, `include/miki/kernel/KernelFactory.h`

### Technical Direction

- **Conditional compilation**: `MIKI_KERNEL_OCCT=ON` CMake option. When OFF, `OcctKernel` source files not compiled. `KernelFactory::Available()` does not include "occt".
- **OCCT API mapping**: `Import("file.step", "step")` → `STEPControl_Reader::ReadFile()` + `TransferRoots()`. `Tessellate(shapeId, quality)` → `BRepMesh_IncrementalMesh(shape, linearDeflection)`.
- **Topology extraction**: `BRep_Tool::Triangulation(face)` → triangle mesh. `BRepAdaptor_Surface::GetType()` → `FaceType` enum. `TopExp_Explorer` for face/edge/vertex enumeration.
- **triangleToFaceMap**: built during tessellation by iterating faces, mapping global triangle indices to `FaceId`.
- **Phase 5 scope**: implement `Tessellate()` and `Import()` (STEP only). Other interface groups (`BooleanOp`, `ExactDistance`, `CreateSketch`, etc.) return `NotImplemented` — implemented in Phase 7b/9.
