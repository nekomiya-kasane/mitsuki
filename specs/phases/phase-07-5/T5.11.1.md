### Technical Direction

- **Demo structure**: create 100K entities with random positions + AABBs. Build BVH + Octree. Each frame: camera frustum query via BVH, range query via Octree, render visible set as colored points or wireframe boxes.
- **ImGui panels**: entity count, archetype breakdown, BVH node count, query timing (frustum ms, ray ms), Octree depth, SystemScheduler execution time.
- **Orbit camera**: reuse existing `OrbitCamera` from demo framework.
- **Integration tests**: headless path (no window) — create entities, build spatial structures, run queries, verify correctness.
