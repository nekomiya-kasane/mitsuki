## Design Decisions

1. **Runtime capability check replaces compile-time guards**: All `#ifdef MIKI_HAS_VULKAN` removed from `BindlessTable.cpp`. `DescBufferStrategy::TryCreate()` uses `IDevice::GetDescriptorBufferInfo()` at runtime — returns `NotSupported` on non-Vulkan backends naturally.
2. **Vulkan::Headers instead of miki::rhi_vulkan**: CMake links `Vulkan::Headers` (type definitions only) instead of the full Vulkan backend library. `BindlessTableDescBuffer.cpp` needs Vulkan types for function pointer loading but doesn't link the backend.
3. **Integration test fix**: `test_resource_manager_integration.cpp` updated to call `SetDefaultSampler()` in SetUp (required by T5.0.3 fix).
