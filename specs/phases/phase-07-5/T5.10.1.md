- **Error model**: `GetFaces(bodyId)` returns empty span if body not found. No exceptions.
- **Thread safety**: read-only after construction — thread-safe for concurrent queries. Mutation (AddBody, AddFace, etc.) on construction thread only.
- **Invariants**: `TopoGraph` is kernel-agnostic — populated from `IKernel::Import` results, never from OCCT types directly. Each face carries `FaceType`. Adjacency is bidirectional.

### Technical Direction

- **BRep hierarchy**: Compound → Solid → Shell → Face → Wire → Edge → Vertex. `TopoNodeType` enum.
- **Adjacency storage**: `TopoGraph` stores face-edge-vertex adjacency as flat arrays with index lists. `GetAdjacentFaces(edgeId) -> span<FaceId>` returns the 1-2 faces sharing an edge.
- **FaceType enum**: `{Planar, Cylindrical, Conical, Spherical, Toroidal, BSpline, Offset, Other}`. Set during import from kernel data.
- **TopoInspector**: utility class for common queries — `GetFacesByType(bodyId, FaceType)`, `GetBoundaryEdges(bodyId)`, `GetAdjacentFaces(faceId)`, `IsManifold(bodyId)`.
- **Namespace**: `miki::topo`
