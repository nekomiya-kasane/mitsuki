### Technical Direction

- **Dual kernel mode**: CLI `--kernel sim|occt` (default: sim). SimKernel generates procedural shapes. OcctKernel loads STEP file (path from CLI `--file`).
- **Pipeline**: `KernelFactory::Create(name)` → `kernel->Import(file)` or `kernel->Tessellate(proceduralShape)` → `PopulateTopoFromTessellation()` → `ComponentPool<MeshComponent>::Attach()` → render via ForwardPass.
- **TopoGraph inspector**: ImGui panel showing face count, edge count, face type distribution (pie chart or bar), vertex count. Click on rendered geometry → ray pick → PrimitiveToFace → highlight face.
- **No-kernel mode validation**: load pre-tessellated mesh data (stub .miki archive concept) without kernel linked — demonstrates pure renderer mode.
