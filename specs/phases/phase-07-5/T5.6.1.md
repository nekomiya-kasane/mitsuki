- **Error model**: `Insert()` returns bool (false if outside world bounds). `Remove()` returns bool (false if not found).
- **Thread safety**: Single-writer, multi-reader after build. Dynamic insert/remove on update thread only.
- **Invariants**: `Query(AABB)` returns all entities whose AABB intersects query box. `Insert(entity, AABB)` places entity in correct leaf. Adaptive subdivision splits when leaf count exceeds threshold (default 8).

### Technical Direction

- **Morton code (Z-order curve)**: 3D position → 30-bit Morton code via bit-interleave. Used for node addressing and cache-coherent traversal.
- **Adaptive subdivision**: each node starts as leaf. When entity count > `maxPerLeaf` (default 8), split into 8 children. Merge when child count drops below threshold.
- **Dynamic insert/remove**: O(log₈ N) per operation. No full rebuild needed for incremental updates.
- **World bounds**: configurable root AABB. Objects outside root are rejected (caller must expand or use unbounded SpatialHash).
