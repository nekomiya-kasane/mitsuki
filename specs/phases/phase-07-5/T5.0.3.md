### Technical Direction

- **Default sampler pattern**: `ResourceManager` stores `std::optional<SamplerHandle> defaultSampler_`. `SetDefaultSampler()` sets it. `CreateTexture(desc, autoBindless=true)` checks `defaultSampler_.has_value()` — if not, return error.
- **Alternative considered**: sampler param in `CreateTexture()` — rejected because it pollutes every call site. Default sampler is set once at init, used many times.
- **Pitfall reference**: `.windsurf/pitfalls.md` documents the null sampler crash

## Design Decisions

1. **Default sampler stored in Impl**: `std::optional<SamplerHandle> defaultSampler` in Pimpl. Set once at init, used for all autoBindless registrations.
2. **Error on missing sampler**: `CreateTexture(desc, autoBindless=true)` returns `ErrorCode::InvalidState` if no default sampler set. Texture is destroyed before returning error to avoid leak.
