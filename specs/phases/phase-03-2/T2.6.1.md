### Technical Direction

- **Dear ImGui integration**: Add `imgui` as third-party dependency (submodule or vendored). Use official renderer backends. `ImGuiBackend` is a thin wrapper that selects the correct `imgui_impl_*` based on backend type.
- **Vulkan backend**: `imgui_impl_vulkan` — requires **ImGui docking branch** with `ImGui_ImplVulkan_InitInfo::UseDynamicRendering = true` + `VkPipelineRenderingCreateInfo{colorAttachmentCount=1, pColorAttachmentFormats=&swapchainFormat}`. No `VkRenderPass` needed (Phase 1a requires Vulkan 1.4 which includes `VK_KHR_dynamic_rendering` core). Descriptor pool: miki-owned pool dedicated to ImGui (separate from T2.3.1 descriptor system).
- **D3D12 backend**: `imgui_impl_dx12` — needs `ID3D12Device`, descriptor heap, root signature.
- **GL backend**: `imgui_impl_opengl3` — needs GL 4.3+ context (already have it).
- **WebGPU backend**: `imgui_impl_wgpu` — needs `WGPUDevice`, `WGPUQueue`.
- **GLFW platform backend**: `imgui_impl_glfw` for input handling in demos. Platform backend is separate from renderer backend.
- **Frame stats**: Simple built-in panel: FPS (smoothed), frame time (ms), backend name, GPU name. Toggle with F3 key.

## Design Decisions

- **Pimpl with namespace-scope base struct**: `ImGuiBackendImpl` defined at namespace scope, `ImGuiBackend::Impl` inherits from it. Allows static free functions to access impl fields without private-access issues.
- **Separate descriptor management**: ImGui manages its own `VkDescriptorPool` (Vulkan) and `ID3D12DescriptorHeap` (D3D12), independent of the engine's descriptor system.
- **Dynamic rendering (Vulkan)**: `UseDynamicRendering = true` with `VkPipelineRenderingCreateInfo` — no `VkRenderPass` needed.
- **VMA promoted to PUBLIC**: `miki_rhi_vulkan` now exports VMA as PUBLIC since `VulkanDevice.h` includes `vk_mem_alloc.h` and is consumed by `miki_render`.
- **Backend-conditional compilation**: `MIKI_BUILD_VULKAN`, `MIKI_BUILD_D3D12`, `MIKI_BUILD_WEBGPU` defines gate backend-specific code in both CMake and C++.
- **F3 toggle**: `ImGui::IsKeyPressed(ImGuiKey_F3)` checked in `BeginFrame()` after `ImGui::NewFrame()`.
- **EMA smoothing**: Frame stats use exponential moving average (alpha=0.1) for FPS and frame time.
