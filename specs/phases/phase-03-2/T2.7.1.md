- **Error model**: `LoadFont(path)` returns `Result<FontId>`. `LoadSystemFont(family, style)` returns `Result<FontId>`. Missing file / unsupported format → error code. FreeType init failure → fatal error in `FontManager::Create()`.
- **Invariants**: `FontId` = opaque uint32_t index into font table. Max loaded fonts: 64 (soft limit, warning log if exceeded). `FontManager` owns all `FT_Face` handles; destroyed on `FontManager` destruction. Move-only Pimpl class.

**Key types**:
```cpp
struct FontMetrics {
    float ascender;      // above baseline (positive)
    float descender;     // below baseline (negative)
    float lineHeight;    // recommended line spacing
    float unitsPerEm;
};
struct GlyphMetrics {
    float advance;       // horizontal advance
    float bearingX;      // left side bearing
    float bearingY;      // top side bearing
    float width;         // bounding box width
    float height;        // bounding box height
};
enum class FontStyle : uint8_t { Regular, Bold, Italic, BoldItalic };
```

### Technical Direction

- **FreeType version**: >= 2.13 for variable font axis support (future-proof). Static lib, vendored or submodule.
- **Font loading**: `FT_New_Face` from file path. Store `FT_Face` + metadata in internal font table. Extract `FontMetrics` from face globals. `GlyphMetrics` queried per-glyph via `FT_Load_Glyph` with `FT_LOAD_NO_BITMAP` (we want outlines for MSDF).
- **System font discovery (Windows)**: `DirectWrite` `IDWriteFontCollection` → enumerate families → match family+style → get file path. Fallback: registry scan `HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts`.
- **System font discovery (Linux)**: **Mandatory** — `fontconfig` (`FcFontList` + `FcFontMatch`) for Linux CI text test execution. Link `libfontconfig` conditionally (`#ifdef __linux__`). Without this, Linux CI cannot discover system fonts and text tests will fail.
- **System font discovery (macOS)**: Stretch goal. `CTFontManagerCopyAvailableFontFamilyNames`.
- **Internal FT_Face access**: `FontManager::GetFreetypeFace(FontId) -> FT_Face` — internal-only (not in public header). Exposed via friend or `detail::` namespace for T2.7.2, T2.7.3 consumption.
- **Bundled fallback font**: Ship one SIL OFL-licensed font (Noto Sans Regular, ~500KB) as embedded resource via `xxd` or CMake `file(READ)`. Used as last-resort fallback and for testing.
- **Thread safety**: Not thread-safe. Single-owner, main-thread only.
