### Technical Direction
- **D3D12**: SRV/UAV descriptor heap. `ID3D12DescriptorHeap` with shader-visible flag. Copy descriptors via `CopyDescriptorsSimple`. Grow = create new heap + copy + swap (deferred destroy old).
- **OpenGL 4.3**: `glBindBufferRange` for SSBO bindings. Texture units via `glBindTextureUnit` (DSA). Limited bindless slots (max ~16 texture units on old HW, ~96 on modern). Fallback: array texture or texture handle (`GL_ARB_bindless_texture` if available).
- **WebGPU**: bind groups + storage buffers. `wgpu::BindGroup` recreation on grow. `maxStorageBufferBindingSize` limit → chunked binding for large tables (>128MB). Each Register produces a bind group entry.
- **Backend dispatch**: T4.2.1a establishes the dispatch pattern (Pimpl or strategy). This task implements the 3 remaining strategy classes.

## Design Decisions

- **No backend-specific BindlessTable files**: BindlessTable uses `IDevice` generic interface (`CreateDescriptorSetLayout/CreateDescriptorSet/UpdateDescriptorSet`). GL and WebGPU already had complete implementations. Only D3D12 was missing.
- **D3D12 descriptor heap emulation**: Added `D3D12DescriptorHeapAllocator` (shader-visible CBV_SRV_UAV heap + CPU-staging heap), `D3D12DescriptorLayoutData`, `D3D12DescriptorSetData`. `UpdateDescriptorSet` creates SRV/CBV/UAV on staging heap then `CopyDescriptorsSimple` to shader-visible.
- **MockDevice DescriptorBuffer removed**: MockDevice was incorrectly claiming `HasDescriptorBuffer() = true`. Removed to prevent BindlessTable from attempting descriptor buffer strategy on Mock.
