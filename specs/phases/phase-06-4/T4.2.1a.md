- **Error model**: `Register()` returns `BindlessIndex` (kInvalidBindless on failure). `Create()` returns `Result<BindlessTable>`.
- **Thread safety**: NOT thread-safe. Resource registration happens on render thread only.
- **Invariants**: table starts at initial capacity (1024). Auto-grows by 2x when full. Old descriptor set/buffer copied to new. BindlessIndex values are stable across grows (indices never change).

### Technical Direction
- **v1 = descriptor set backend**: all tiers use traditional descriptor sets. Vulkan: `vkAllocateDescriptorSets` with `VK_DESCRIPTOR_TYPE_COMBINED_IMAGE_SAMPLER` / `VK_DESCRIPTOR_TYPE_STORAGE_BUFFER` / `VK_DESCRIPTOR_TYPE_SAMPLED_IMAGE`. D3D12: SRV/UAV descriptor heap. GL: `glBindBufferRange` + SSBO. WebGPU: bind groups + storage buffers.
- **Tier dispatch**: `BindlessTable` internally dispatches to per-backend strategy. Public API is tier-agnostic.
- **Auto-grow**: when table is full, allocate new descriptor set with 2x capacity, copy all existing descriptors, swap. Old set destroyed after kMaxFramesInFlight frames (deferred).
- **Slot reuse**: freed BindlessIndex values are recycled via internal free-list. No index inflation.
- **Global binding**: BindlessTable's descriptor set is bound at set 0 (or configurable set index) for all draw/dispatch calls. One bind per frame.

## Design Decisions

- Pimpl pattern: `BindlessTable::Impl` holds all state (device ptr, descriptor handles, free-list, alive flags). Public header has zero backend includes.
- Single descriptor set with 3 bindings: CombinedImageSampler (textures), StorageBuffer (buffers), StorageTexture (UAVs). Each binding has `count = capacity` for array indexing.
- Auto-grow: when AllocateSlot exhausts free-list and all capacity used, Grow() doubles capacity, creates new layout+set, destroys old (immediate in Mock, deferred in real backends later).
- Free-list is LIFO (stack): `freeList_.push_back()` / `pop_back()`. O(1) alloc/free.
- `Size()` counts alive flags (O(n)) — acceptable for debug/test use. Hot path uses `liveCount` field.
- MockDevice doesn’t support `IDevice::CreateOwned(Mock)` — tests construct `MockDevice` directly with leak check disabled.
