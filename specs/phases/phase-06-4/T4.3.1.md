- **Error model**: `Register()` returns `BDAPointer` (null address = failure). `Validate()` returns `bool`.
- **Thread safety**: NOT thread-safe. Registration on render thread only.
- **GPU constraints**: `BDAPointer` is `alignas(16)`, 16 bytes, suitable for push constants and SSBO.
- **Invariants**: FNV-1a checksum computed from address bytes. Debug builds validate checksum before GPU use. Release builds skip validation (zero overhead).

### Technical Direction
- **BDAPointer layout**: `{ uint64_t address; uint32_t checksum; uint32_t size; }` = 16B. `address` is the GPU virtual address. `size` is the buffer byte size (for bounds checking in debug). `checksum` = FNV-1a of `address` bytes (8 bytes -> 32-bit hash).
- **Per-backend address query**: Vulkan: `vkGetBufferDeviceAddress` (requires `VK_BUFFER_USAGE_SHADER_DEVICE_ADDRESS_BIT` on buffer). D3D12: `GetGPUVirtualAddress()`. GL: emulated — return SSBO binding index encoded as address. WebGPU: emulated — return storage buffer binding index. Mock: return sequential fake addresses.
- **Registration table**: `std::unordered_map<BufferHandle, BDAPointer>` for CPU-side lookup. O(1) register/remove/query.
- **Slang integration**: BDA accessed in Slang via `ByteAddressBuffer` or `StructuredBuffer<T>` with explicit address. Push constant carries `BDAPointer.address` for per-draw vertex buffer access.
- **DeviceFeature::BufferDeviceAddress**: must be enabled (Vulkan 1.2 core, always available on Tier1/2).

## Design Decisions

- `BDAPointer` is a plain struct (not Pimpl) — 16B value type for push constants and SSBO. `alignas(16)` for GPU compatibility.
- `ComputeBDAChecksum` is `constexpr` FNV-1a over 8 address bytes — usable at compile-time for static validation.
- `MakeBDAPointer` helper constructs a validated pointer in one call.
- `BDAManager` uses Pimpl. Internal `QueryAddress` currently returns monotonic fake addresses (0x1000, 0x2000, ...) — sufficient for Mock backend. Real Vulkan/D3D12 BDA requires extending `IDevice` with `GetBufferDeviceAddress()` (deferred to Phase 6a when mesh shader BDA vertex fetch is implemented).
- Registration table: `std::unordered_map<BufferHandle, BDAPointer>` with custom hash on `_value`.
- 17 tests (spec: 10) — added `ChecksumDifferentForDifferentAddresses`, `ValidateFailNullAddress`, `MakeBDAPointerSetsAllFields`, `Equality`, `RegisterInvalidBufferReturnsNull`, `GetPointerUnregisteredReturnsNull`, `MultipleRegistrations`.
