- **Error model**: `Track/Release` are void (fire-and-forget bookkeeping). `GetPressureState()` is noexcept query. Eviction callbacks invoked synchronously when pressure changes.
- **Thread safety**: NOT thread-safe. Updated on render thread via `ResourceManager::Tick`.
- **Invariants**: per-category byte counts always >= 0 (underflow is assert in debug). Pressure state transitions are monotonic within a frame (Normal->Warning->Critical->OOM, never backwards within single `Update()`).

### Technical Direction
- **Budget categories**: `enum class BudgetCategory : uint8_t { Geometry, Texture, Staging, AccelStruct, RenderTarget, Misc, Count_ }`. Each category tracks `{usedBytes, budgetBytes, peakBytes}`.
- **Pressure state machine**: `enum class PressureState { Normal, Warning, Critical, OOM }`. Thresholds configurable (default: Warning=70%, Critical=85%, OOM=95% of total budget, per `rendering-pipeline-architecture.md` §20.5). Hysteresis: 5% band to prevent oscillation.
- **LRU eviction callbacks**: `RegisterEvictionCallback(BudgetCategory, EvictionFn)`. When pressure transitions to Critical, invoke callback with target bytes to free. Callback is a `std::move_only_function<void(uint64_t bytesToFree)>`.
- **VMA integration**: Vulkan path queries `VmaBudget` via `vmaGetBudget`. `budgetBytes` = `VmaBudget::budget` (OS-reported). `usedBytes` = sum of tracked per-category bytes (not VMA's total — we need per-category breakdown).
- **Hard limit mode**: `SetHardLimit(category, bytes)` — for testing/validation. Create fails if allocation would exceed limit.

## Design Decisions

- Value type (no Pimpl) — MemoryBudget is small, no GPU resources, direct member storage.
- Pressure thresholds compared in integer space (`threshold * budget` bytes) to eliminate float precision issues at exact boundaries (e.g., 850/1000 = 0.85 must equal `criticalThreshold`).
- Downward transitions cascade in a single `Update()` call via while-loop — e.g., Critical→Normal in one call if usage drops below all hysteresis bands.
- `std::function` used instead of `std::move_only_function` (C++23 not available in COCA libc++).
- HardLimit clamps Track() silently (no error return) — caller doesn’t need to handle failure.
- 15 tests including hysteresis oscillation prevention, eviction callback ordering, and full pressure cycle integration.
