- **Error model**: `CreateTexture/Buffer/...` returns `Result<ResourceHandle>`. `Destroy()` is fire-and-forget (deferred). `GetRhiHandle()` returns `std::optional` (nullopt if handle stale).
- **Thread safety**: NOT thread-safe. All operations on render thread.
- **Invariants**: deferred destruction: `Destroy(handle)` pushes to per-frame free list. Actual RHI `Destroy*()` called after `kMaxFramesInFlight` frames via `Tick()`. ResourceHandle generation prevents stale access.

### Technical Direction
- **Facade pattern**: `ResourceManager` wraps `IDevice` + `SlotMap` + `BindlessTable` + `BDAManager` + `StagingRing`. Single API for the application layer. No direct `IDevice::CreateTexture/Buffer` calls from scene/render code after Phase 4.
- **Internal `ResourceEntry`**: `{ rhi::TextureHandle or rhi::BufferHandle (variant); BindlessIndex; BDAPointer; ResourceType; uint32_t frameDestroyed; }`. Stored in `SlotMap<ResourceEntry>`.
- **Deferred destruction**: `Destroy(ResourceHandle)` marks entry as dead + records `frameDestroyed`. `Tick(currentFrame)` scans the dead list and calls actual `IDevice::Destroy*()` when `currentFrame - frameDestroyed >= kMaxFramesInFlight`.
- **Auto-register**: deferred to T4.5.1b. T4.5.1a does not depend on BindlessTable or BDAManager.
- **Budget callback**: deferred to T4.5.1b. T4.5.1a does not call MemoryBudget.

## Design Decisions

- Pimpl with `SlotMap<ResourceEntry>` as internal storage. `ResourceEntry` holds `std::variant<TextureHandle, BufferHandle, SamplerHandle>` + metadata.
- `Create()` simplified to accept only `IDevice&` for T4.5.1a scope. T4.5.1b will add BindlessTable/BDA/StagingRing params.
- Deferred destruction: `Destroy()` sets `alive=false`, decrements liveCount, pushes to `pendingDestroys`. `Tick(frameIndex)` tags new entries with frameDestroyed, then destroys entries where `frameIndex - frameDestroyed >= kMaxFramesInFlight`.
- `GetBindlessIndex`/`GetBDAPointer` return default (invalid) values in T4.5.1a — wired in T4.5.1b.
- Destructor cleans up both live and pending-deferred resources.
- 18 tests (spec: 13) — added `CreateSamplerReturnsValidHandle`, `GetBufferHandleRoundTrip`, `GetTextureHandleWrongType`, `IsAliveReturnsTrueForLive`, `DoubleDestroyIgnored`.
